A simple **Point of Sale (POS)** application written in **Python**, designed for managing basic sales operations with a clean graphical interface.

---

## 🛒 Overview

This project implements a desktop POS system where users can add products, process sales, calculate totals, and generate receipts. Perfect for small shops, food stalls, or as a learning project for GUI-based Python applications.

---

## 📋 Features

- GUI-based interface for sales processing  
- Product selection and pricing  
- Automatic calculation of totals and change  
- Receipt generation and display  
- Modular code with separate frames for UI components  
- Lightweight and easy to extend
